核心能力

可视化设计器的核心功能模块。

🎨 可视化设计器

EditorView 为核心画布，支持拖拽组件到画布、选中编辑、Schema 树浏览、实时预览。左侧面板展示组件分组，右侧属性面板 Schema 驱动配置。

⚡ 事件引擎

统一的 eventEngine 支持 5 种触发类型（click/change/focus/blur/close）、15+ 种动作类型（show/hide/open-dialog/switch-tab/set-value/submit/reset/emit）、条件表达式沙箱执行。

🔗 联动系统

6 种联动类型：visible / disabled / required / options / set-value / reset-fields。通过 watchFields 监听字段变化，条件表达式驱动组件状态联动。

📦 拖拽系统

useDrag composable 统一管理拖拽状态，支持从组件面板拖入画布、画布内排序、跨布局容器拖动。dragStore 管理全局拖拽状态。

🔄 撤销/重做

editorStore 管理历史栈，主画布和弹窗编辑器独立管理。支持快照恢复，clipboard 剪贴板操作。

🌐 微前端集成

通过 vite-plugin-qiankun 支持 qiankun 子应用模式（/child/schemaForm/ 前缀），同时支持 microapp 子应用集成。

🔌 数据源配置

OptionsApiConfigDialog 支持 API 模式和字典码模式加载动态选项，JSONPath 路径提取，响应格式标准化。

📋 属性面板

Schema 驱动的右侧属性配置面板，通过 useRightPanelConfig 动态渲染。支持 BorderEditor、SpacingEditor、ConditionBuilder 等 10+ 专用编辑器。

快速开始

# 安装依赖
pnpm install

# 启动开发服务器（端口 5173）
pnpm dev:web

# 构建生产版本
pnpm build:web

# 类型检查
cd packages/web && npx vue-tsc -b --noEmit

项目结构

│ ├── components/
