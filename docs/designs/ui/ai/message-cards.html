已为您创建用户注册表单，包含以下 4 个字段：

  • 用户名 - 文本输入，必填，3-20 个字符
  • 邮箱 - 邮箱输入，必填，需验证格式
  • 手机号 - 手机号输入，必填，11 位数字
  • 密码 - 密码输入，必填，至少 8 位，需包含字母和数字

您可以点击 "确认发布" 将表单发布到生产环境，或者点击 "修改" 进行调整。

以下是表单配置的 JSON 结构：

{
  "type": "form",
  "fields": [
    { "name": "username", "type": "input", "required": true },
    { "name": "email", "type": "email", "required": true },
    { "name": "phone", "type": "phone", "required": true },
    { "name": "password", "type": "password", "required": true }
  ]
}

您可以使用 validate_schema 工具来校验配置是否正确。

提示：您可以通过自然语言描述来修改表单字段，例如"添加一个年龄字段"或"将手机号改为可选"。
