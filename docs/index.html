子项目

平台由四个核心模块组成，各司其职，通过 Schema 和 API 协作。

表单编辑器 (Editor)

Schema 驱动的可视化自由布局表单设计器与运行时渲染引擎。支持拖拽编排、属性配置、事件联动、图表集成。

  • 49 个 Widget 组件，8 大分组
  • 三栏布局：部件库 + 画布 + 属性面板
  • 14 种事件动作，6 种联动类型
  • 自由布局：绝对定位 + 拖拽手柄
  • 9 种 ECharts 图表组件
  • qiankun / micro-app 微前端集成
Vue 3.5 Element Plus Pinia ECharts
查看文档 →

流程设计器 (Flow)

BPMN 2.0 流程设计器与引擎，支持可视化流程编排、BPMN XML 导入导出、流程监控。

  • 25 种 BPMN 节点类型
  • 事件 / 任务 / 网关 / 子流程 四大分类
  • BPMN 2.0 XML 导入导出
  • dagre 自动布局算法
  • 流程监控面板
  • 表单节点嵌入预览
Vue Flow BPMN 2.0 dagre
查看文档 →

AI 助手 (AI App)

AI 智能助手应用，通过自然语言对话驱动表单 Schema 和流程图的生成，支持多 Agent 协作。

  • 3 种 Agent：Auto / Editor / Flow
  • SSE 流式响应
  • 多模态输入（图片、PDF、Word）
  • RAG 语义检索
  • 版本管理与对比
  • Micro-app 嵌入 Editor/Flow
LangGraph SSE RAG

后端服务 (Server)

Koa.js + MongoDB 后端 API 服务，提供 Schema CRUD、认证、AI Agent 图编排、健康检查。

  • Schema 管理：CRUD / 发布 / 版本
  • JWT + bcryptjs 认证
  • AI Agent：LangGraph 图编排
  • 数据源 / 字典 / Mock 数据
  • Zod 请求校验
  • 健康检查：API + DB ping

共享包

跨模块复用的核心能力。

@schema-form/shared-ai

AI 元数据与 prompt 构建。从 Widget/Flow 节点定义提取，保证 AI 知识单一数据源。

@schema-form/ai-sdk

独立可复用的 Agent SDK，与 LangGraph 解耦，支持 BaseAgent、ToolRegistry、PromptBuilder。

@schema-form/flow-shared

流程引擎共享层，BPMN 2.0 类型系统、XML 导入导出引擎。

@schema-form/socket

WebSocket 实时通信，AI 协作同步。
