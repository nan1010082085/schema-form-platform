  • 9 种 ECharts 图表组件
  • AI 生成 Schema 一键应用到画布

图流转路径

START → thinker → editor | flow | page | general
editor/flow/page ↔ allTools → afterTools
afterTools → thinker（继续链）或 summarizer（结束）

任务链自动编排：联动生成时 Thinker 将复杂需求拆分为多步，依次调度不同专家 Agent，最后由 Summarizer 整合结果。

Agent 协作机制

request_collaboration 工具

Agent 可主动请求其他专家协助。例如 Flow Agent 在生成流程时发现缺少表单，可请求 Editor Agent 生成。

afterTools 节点

扫描消息中的协作请求，写入 state.collaborationRequest，由 Thinker 将协作步骤插入任务链。

共享包

跨模块复用的核心能力，AI 贯穿其中。

@schema-form/shared-ai

AI 元数据与 prompt 构建。从 Widget/Flow 节点定义提取 metadata.json，promptBuilder 动态生成 system prompt，保证 AI 知识单一数据源。

@schema-form/ai-sdk

独立可复用的 Agent SDK，与 LangGraph 解耦，支持 BaseAgent、ToolRegistry、PromptBuilder。

@schema-form/flow-shared

流程引擎共享层，BPMN 2.0 类型系统、XML 导入导出引擎。

@schema-form/socket

WebSocket 实时通信，AI 双向同步（Editor/Flow 协作）。
