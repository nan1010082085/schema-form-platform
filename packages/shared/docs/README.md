# Shared 文档

共享包集合，提供跨项目复用的能力。

## 包列表

| 包 | 路径 | 说明 |
|---|---|---|
| `@schema-form/shared-qiankun` | `shared/qiankun/` | 微前端生命周期、状态、事件、配置 |
| `@schema-form/shared-components` | `shared/components/` | 共享 Vue 组件 |
| `@schema-form/shared-config` | `shared/config/` | Vite/Element Plus 配置 |
| `@schema-form/shared-stores` | `shared/stores/` | 共享 Pinia Store |
| `@schema-form/shared-utils` | `shared/utils/` | 工具函数（API 客户端、认证等） |
| `@schema-form/shared-styles` | `shared/styles/` | 共享样式（主题、变量、组件样式） |
| `@schema-form/socket` | `shared/socket/` | WebSocket 通信 |
