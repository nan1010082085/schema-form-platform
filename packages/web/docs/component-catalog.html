🔗 联动配置 (Linkage)

6 种联动类型

类型触发条件真时行为假时行为
visiblecondition=true显示字段隐藏 + elseValue回退
disabledcondition=true禁用字段解除禁用
requiredcondition=true设为必填取消必填
optionscondition=true切换选项（thenOptions/thenApi）恢复默认 + elseValue回退
set-valuecondition=true设值（thenValue字面值/valueSource字段复制）elseValue回退
reset-fieldscondition=true重置targetFields到defaultValue—

多字段联动示例 — Checkbox → 单字段控制

{"type":"checkbox","field":"features","label":"功能选择","options":[ {"label":"高级审批","value":"advanced"}, {"label":"加急处理","value":"urgent"} ], "linkages":[ {"type":"visible","watchFields":["features"],"condition":"values.features && values.features.includes('advanced')","elseValue":""}, {"type":"set-value","watchFields":["features"],"condition":"values.features && values.features.includes('urgent')", "thenValue":"加急模式已启用","valueSource":undefined} ]}

联动配置公共属性

属性类型说明
typeLinkageTypevisible|disabled|required|options|set-value|reset-fields
watchFieldsstring[]监听的字段名数组
conditionstring | Function字符串表达式（沙箱eval）或函数
elseValueFormFieldValue条件 false 时回退值
thenOptionsDictItem[]仅options类型：条件true时替换选项
thenApiSchemaApiConfig仅options类型：条件true时替换API
thenValueFormFieldValue仅set-value类型：条件true时字面值
valueSourcestring仅set-value类型：复制值的源字段
targetFieldsstring[]仅reset-fields类型：要重置的字段数组

⚠️ 循环依赖自动检测降级 — useLinkage 中 DFS 检测，循环字段回退到默认状态

🌐 API 数据配置 (SchemaApiConfig)

属性类型默认值版本说明
urlstring—S7API地址（必填）
method'get' | 'post'getS7HTTP方法
paramsRecord<string,unknown>—S7查询/body参数，支持 ${fieldName} 模板
dataPathstring—S16dot-notation数据路径（如result.records），回退链: data>list>rows>items>records
labelKeystringlabelS7映射到DictItem.label的键
valueKeystringvalueS7映射到DictItem.value的键
childrenKeystring—S17树形数据子节点键（保留嵌套结构不拍平）
ttlnumber0S17缓存TTL毫秒，0=永不过期
immediatebooleantrueS7挂载时立即加载
dictCodestring—S7字典码（优先于url，从global.dictMap查找）
